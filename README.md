How to run each version:

- To run the each program, navigate to the Abstract-vs-Interface-Design-Debate folder in your file explorer.

- Open your Terminal and navigate to the project using this command (with your personal directory):

```
   cd C:Yourdirectory/Abstract-vs-Interface-Design-Debate
```

- Once you have successfully navigated to this folder, run this command:

```
javac shapes\abstractdesign\*.java shapes\interfacedesign\*.java shapes\interfaces\*.java
```

From here you can choose which program to run.

- For abstractdesign run the command:

```
java shapes.abstractdesign.Main
```

- For interfacedesign run the command:

```
java shapes.interfacedesign.Main
```


Why this comparison matters in real-world design:

Each version of this project has its own benefits on the coding world.

- Abstract methods are used for making subclasses that all implement the same methods and run inherently the same way.
- Instances are more like blueprints that can be implemented in certain files to easily set up what the file should do and how it should behave
